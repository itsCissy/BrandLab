Code Block

Mono font, light surface, fixed syntax-highlight palette: keyword blue, string green, comment grey, number orange.

TypeScript 
// Spin up an agent and stream its output
import { createAgent } from "@agentic/sdk";

const agent = await createAgent({
  model: "claude-3-5-sonnet",
  temperature: 0.2,
  maxTokens: 4096,
});

for await (const token of agent.stream("Summarize this paper")) {
  process.stdout.write(token);
}
